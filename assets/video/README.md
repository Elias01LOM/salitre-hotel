# Recursos de Video -- Hotel Salitre

## Archivos requeridos

| Archivo | Formato | Dimensiones | Peso maximo | Notas |
|---------|---------|-------------|-------------|-------|
| hero-bg.mp4 | MP4 (H.264) | 1920x1080px | 5MB | Sin audio, loop, autoplay |
| hero-bg.webm | WebM (VP9) | 1920x1080px | 4MB | Fallback para Firefox/Chrome |

## Especificaciones tecnicas

- Sin pista de audio (muted en el tag HTML)
- Duracion recomendada: 10-20 segundos
- Loop continuo
- Caracteristicas: paisaje costero, naturaleza, coworking
- El overlay oscuro (rgba(44,53,57,0.58)) se aplica via CSS

## Referencia

- Documentacion Seccion 09.1 (Video Hero)
- El usuario proporcionara estos recursos
- Mientras no exista, se muestra fondo solido (var(--color-text)) como fallback
