# Hotel Salitre — Contexto de Arquitectura para Cursor

## Qué es este proyectovdvd

Aplicación web para **Hotel Salitre**, un boutique hotel costero para nómadas digitales y trabajadores remotos. Stack: PHP puro + MySQL (PDO) + HTML5 + CSS3 + JS vanilla. Sin frameworks. Sin librerías externas salvo Google Fonts.

Tagline: *"Sal de la oficina. No del trabajo."*
Público: nómadas digitales, freelancers, equipos en sprint remoto.
Los tres pilares del hotel: **Conectividad** (fibra óptica garantizada) · **Ergonomía** (escritorios reales, monitores externos) · **Diseño sin fricción** (minimalismo funcional).

### Tablas y función

| Tabla | Funcion |
|-------|---------|
| clientes | Huespedes registrados en el sitio publico |
| staff | Administradores del panel (acceso solo por URL directa) |

### Relaciones

- reservas.cliente_id -> clientes.id (ON DELETE CASCADE)
- reservas.espacio_id -> espacios.id (ON DELETE RESTRICT)

### Conexion — config/database.php

```php
function getDB(): PDO {
  static $pdo = null;
  if ($pdo === null) {
    $dsn = "mysql:host=localhost;dbname=salitre_db;charset=utf8mb4";
    $pdo = new PDO($dsn, 'root', '', [
      PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
      PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
      PDO::ATTR_EMULATE_PREPARES   => false,
    ]);
  }
  return $pdo;
}
```

## Comportamiento del carrito

El carrito NO procesa pago. Solo genera una solicitud con estado "pendiente".

SESSION["carrito"] contiene:
```php
[
  "espacio_id"    => int,
  "fecha_entrada" => "YYYY-MM-DD",
  "fecha_salida"  => "YYYY-MM-DD",
  "noches"        => int,
  "precio_noche"  => float,
  "precio_total"  => float   // precio_noche * noches
]
```

Calculo del desglose en carrito/index.php:
```php
$subtotal      = $carrito["precio_total"];
$fee_limpieza  = LIMPIEZA_FEE;  // 25.00 definida en constants.php
$iva           = $subtotal * IVA; // 0.16 definida en constants.php
$total         = $subtotal + $fee_limpieza + $iva;
```

El boton final dice "Solicitar Reserva" — POST a procesar_reserva.php — INSERT en reservas con estado="pendiente" — redirect a confirmacion.php con el ID de reserva.
